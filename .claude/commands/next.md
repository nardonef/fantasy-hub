Figure out what to work on next and start building it.

1. Read `docs/progress.md` to see what's done and what's remaining
2. Read `CLAUDE.md` for project conventions and current phase
3. Identify the highest-impact unchecked item(s) considering:
   - Dependencies (what unblocks other work?)
   - Current phase (V1 items before V2)
   - Completeness (finish a section before starting a new one)
4. State what you're going to work on and why
5. Build it
6. After completing, update `docs/progress.md` to check off what you finished
